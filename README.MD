# TELEGRAM FOOL BOT
## HOW TO RUN
* git clone https://github.com/sverlova/tg_fool_bot.git
* cd tg_fool_bot
* python3 -m venv env
* source env/bin/activate
* export PYTHONPATH=$PYTHONPATH:/$PWD
* python3 -m pip install -r requirements.txt
* python3 fool_bot/main.py tocken
